# Weather Data Visualizer - Yash Verma

This project is a weather data analysis mini project made for the course **Programming for Problem Solving using Python**.

It performs:
- Data Loading
- Data Cleaning
- Statistical Analysis
- Grouping & Aggregation
- Visualizations using Matplotlib
- Exporting Cleaned Data

---

##  Files Included
- `weather_analysis.py` – Main Python script
- `weather.csv` – Raw dataset
- `cleaned_weather_data.csv` – Cleaned dataset
- `daily_temperature.png` – Line chart
- `monthly_rainfall.png` – Bar chart
- `humidity_vs_temperature.png` – Scatter plot
- `combined_plots.png` – Combined figure
- `report.pdf` – Full project report (or report.docx)

---

##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

##  Visualizations
1. Daily Temperature Line Chart  
2. Monthly Rainfall Bar Chart  
3. Humidity vs Temperature Scatter Plot  
4. Combined Figure with Subplots  

---

##  How to Run the Project

### 1. Install Dependencies

pip install pandas numpy matplotlib


### 2. Run the Python File


python weather_analysis.py
